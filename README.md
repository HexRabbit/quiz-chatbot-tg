# CCNS Quiz Bot - Telegram Client

## Usage
Install dependencies by pipenv
```
$ pipenv install
```

Fill in telegram bot token & [quiz-server](https://github.com/ccns/quiz-server) host ip to `.config` then start the bot
```
$ pipenv run ./app.py
```
